# 음식 찾기 페이지
## 개발 언어: HTML, CSS, JavaScript

키워드 입력 시 관련된 요리를 찾아서 정보를 가져와서 보여주는 페이지

https://www.themealdb.com 에서 조회할 데이터에 맞는 방법(전체 조회, 특정 ID로 조회, 랜덤 조회)으로 API를 불러옴

### 1. 검색 결과가 있는 경우 
* 검색어를 보여주는 문장을 띄움
* 조회된 데이터들을 전부 보여줌(이미지, hover 시 이름 보임)
* 특정 음식 클릭 시 정보 출력
